# 뉴스레터 에셋 폴더 (임시 보관함)

'운영부 마법사' 주간 뉴스레터에 실을 사진·영상을 여기에 넣고 업로더를 실행합니다.

> ⚠️ **이 레포는 공개 GitHub 레포입니다** — 직원 사진은 레포에 커밋하지 않습니다.
> 이 폴더의 미디어 파일은 .gitignore 처리되어 있고, 실제 서빙은
> Supabase Storage `newsletter` 공개 버킷에서 합니다.

## 사용법

1. 사진(jpg/png/webp)·영상(mp4)을 이 폴더에 넣는다
   — **파일명이 그대로 뉴스레터 캡션이 됩니다** (예: `발표하는 승철 부장님.jpeg`)
2. 업로더 실행 (리사이즈 + Storage 업로드 + 캡션 매핑):
   ```bash
   node scripts/team-briefing/upload-assets.mjs
   ```
3. 금요일 발행 시 draft API가 **최근 7일 내 업로드분**을 자동 수집해
   커버(첫 장) + 이번 주 앨범(최대 6장) + 영상으로 싣습니다.
4. 업로드 후 로컬 원본은 지워도 됩니다.
