# 뉴스레터 에셋 폴더

주간 브리핑 뉴스레터(/r/briefing/[token])에 쓸 사진·영상을 이 폴더에 넣습니다.
파일은 배포 후 `https://<도메인>/newsletter/<파일명>` 으로 서빙됩니다.

## 권장 규약 (추후 자동 연동 예정)

- `cover-<호수>.jpg` — 해당 호 상단 커버 (예: `cover-1.jpg`)
- `cover-default.jpg` — 호별 커버가 없을 때 기본 커버
- `section-<이름>-*.jpg` — 섹션 삽입용 (예: `section-ai-01.jpg`)
- 영상은 유튜브/스트림 링크 사용 권장 (파일 직접 첨부보다 용량·재생 안정)

이미지 권장 크기: 가로 1280px 이하, 1MB 이하 (뉴스레터 본문 폭 640px).
